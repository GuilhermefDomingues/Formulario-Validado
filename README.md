# Form Validation JS

Projeto desenvolvido com HTML, CSS e JavaScript que implementa validação de formulários no lado do cliente (Client Side Validation). O sistema verifica os dados inseridos pelo usuário antes do envio do formulário.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Validação de Nome
- Validação de E-mail
- Validação de Senha
- Validação de Select
- Validação de Mensagem
- Verificação de Campos Obrigatórios
- Feedback para o Usuário
- Manipulação do DOM

## 📂 Estrutura do Projeto

```bash
Form-Validation-JS/
│
├── index.html
├── style.css
├── FormJS.js
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html`.
4. Teste os campos e validações do formulário.

Praticar validações de formulários utilizando JavaScript puro, simulando comportamentos comuns encontrados em aplicações web profissionais.
